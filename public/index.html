Draw & Guess – Skribbl Clone

Rules of the Game

  1. This is a multiplayer drawing & guessing game for 2–12 players per room. Play in real-time via browser!
  2. Objective: Draw the secret word so others can guess it. Score points by guessing correctly (faster = more points) or drawing well (everyone guesses your word).

How to Play

  1. Join/Create Room: Enter name & 6-letter code. Share code with friends. Game auto-starts at 2+ players.
  2. Rounds: 6 total, each turn has a new drawer. Info shows top-right.
  3. Draw: Pick a word from 3 options, draw (no typing/speaking your word).
  4. Guess: Type guesses in chat. Correct = points (faster=more). Drawer gets bonus for each correct guess.
  5. Hints & Timer: Word length is shown as blanks. 75-second timer. Time up → reveal word → next round.
  6. End Game: After 6 rounds, leaderboard pops up. Highest score wins!
